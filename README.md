# Retail Sales Analytics

## Objective

Analyze 9,994 retail transactions to identify trends in revenue, profitability, customer behavior, and regional performance using PostgreSQL.

## Skills Demonstrated

- SQL
- Data Exploration
- Aggregations
- JOINs
- CTEs
- Window Functions
- Business Analysis

## Key Insights

- Technology generated the highest sales revenue.
- California was the most profitable state.
- Several subcategories generated negative profit despite strong sales.
- The top 10 customers contributed a significant portion of total revenue.
